# README.md - Описание проекта
## Каталог фильмов

Консольное приложение для управления каталогом фильмов с хранением данных в JSON.

## Запуск программы

1. Клонируйте репозиторий
2. Установите Python 3.8+
3. Запустите: `python app.py`

## Функциональность
- Просмотр списка фильмов
- Добавление новых фильмов
- Отметка просмотренных
- Поиск по году выпуска
